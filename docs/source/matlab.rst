Usage with MATLAB
==================

The Python Anesthesia Simulator (PAS) can also be used from MATLAB. This is made possible by importing a python object
in MATLAB. The following sections explain how to set up your MATLAB environment to use PAS, and demonstrate how to use it in both an open-loop and a closed-loop simulation.
